## 22 种艺术指导风格 (22 Art Direction Styles)

1. **极简风格 (Minimalist)** —— 大面积留白、单一视觉焦点元素、1-2 种配色、干净的无衬线体
2. **大胆文字 (Bold Typography)** —— 文字即设计；以超大字号、极具表现力的字体排版填满画布
3. **渐变 / 晕染 (Gradient / Color Wash)** —— 平滑的色彩过渡、网格渐变、色彩融合
4. **图片基底 (Photo-Based)** —— 全屏大图基底配合文字叠层；多采用展现生活方式的主视觉图
5. **插画 / 手绘 (Illustrated / Hand-Drawn)** —— 定制化插画、专属图标设计、体现匠人艺术感
6. **几何 / 抽象 (Geometric / Abstract)** —— 以几何形状、线条、网格作为主要视觉元素
7. **怀旧 / 复古 (Retro / Vintage)** —— 沧桑做旧质感、柔和色调、衬线体、半调网点效果
8. **毛玻璃 (Glassmorphism)** —— 磨砂玻璃面板、背景模糊、微妙的边缘发光效果
9. **3D / 雕塑感 (3D / Sculptural)** —— 具有深度的 3D 渲染物体、阴影效果；以产品为中心
10. **霓虹 / 赛博朋克 (Neon / Cyberpunk)** —— 深色背景、明亮的霓虹灯点缀、高对比度效果
11. **双色双调 (Duotone)** —— 对照片进行双色调滤镜处理；在图片上叠加醒目的品牌颜色
12. **社论 / 杂志 (Editorial / Magazine)** —— 重度依赖网格排版、金句提炼、新闻纪实构图风格
13. **拼贴 / 混搭 (Collage / Mixed Media)** —— 剪纸肌理、照片剪裁素材、多层次重叠组合
14. **复古未来主义 (Retro Futurism)** —— 对太空时代的怀旧、金属铬质感、色彩渐变、乐观主义色彩
15. **反设计 / 表现主义 (Expressive / Anti-Design)** —— 混乱的布局、混搭字体、故意使用“错误”的构图
16. **数码萌系 / 卡哇伊 (Digi-Cute / Kawaii)** —— 圆润的形状、马卡龙渐变色、像素画、趣味卡通角色
17. **触觉感官 (Tactile / Sensory)** —— 蓬松/软糯纹理、超真实材质感、浮雕工艺效果
18. **数据 / 信息图 (Data / Infographic)** —— 数据指标处于视觉核心、图表、数字作为主视觉
19. **暗黑 / 情绪调性 (Dark Mode / Moody)** —— 接近纯黑的背景、浓郁的宝石色调、高对比度
20. **扁平 / 纯色 (Flat / Solid Color)** —— 单一底色、干净的扁平化图标、不包含任何渐变
21. **自然 / 有机 (Nature / Organic)** —— 大地色系、植物花卉图案、传达可持续发展的品牌形象
22. **动态就绪 (Motion-Ready / Kinetic)** —— 专为动画设计；包含多层元素、可循环播放

## 设计原则 (Design Principles)

### 视觉层级（三区法则 - 3-Zone Rule）
- **顶部 (Top)**：展示 Logo 或核心价值主张 (Value prop)
- **中部 (Middle)**：承载辅助说明信息 + 核心视觉图形
- **底部 (Bottom)**：行动呼吁区域 (CTA，包含按钮/二维码/网址)

### 安全区域 (Safe Zones)
- 核心内容必须保留在画布的中心 70-80% 区域内
- 避免将文字/CTA 放置在距离边缘 50-100px 的范围内
- YouTube 频道：2560 × 1440px 尺寸中，安全区域为 1546 × 423px
- Meta/Instagram：居中 80% 以避开平台自带的 UI 层遮挡

### 行动呼吁规范 (CTA Rules)
- 每个横幅有且仅有一个行动呼吁 (CTA)
- 与背景色彩保持高对比度
- 放置在右下角（视觉的终点区域）
- 针对移动端，触控区域高度至少 44px
- 动词推荐：使用 “Get (获取)”、“Start (开始)”、“Download (下载)”、“Claim (领取)” 等

### 字体排印规范 (Typography)
- 每个横幅最多使用 2 种字体
- 在数字设备上，正文最小 16px，标题大于等于 32px
- 颜色对比度至少达到 4.5:1
- 每行最多 7 个单词，广告内容最多 3 行

### 图文比重 (Text-to-Image Ratio)
- 广告类：文字占比需低于 20%（Meta 广告对高文字占比有惩罚）
- 社交媒体封面：图文比为 60/40
- 印刷品：针对 3-5 米的观看距离，标题字号需在 70pt 以上

### 印刷技术参数 (Print Specs)
- 分辨率最低 300 DPI (大型户外广告最低 150 DPI)
- 四周留出 3-5mm 的出血线 (Bleed)
- 采用 CMYK 色彩模式
- 遵守每增加 1 英尺观看距离，字号增加 1pt 的规则

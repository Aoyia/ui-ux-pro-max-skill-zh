# 核心信息框架 (Messaging Framework)

## 框架结构

```
使命 (我们为何存在)
    ↓
愿景 (我们将走向何方)
    ↓
价值主张 (我们提供什么)
    ↓
定位陈述 (我们有何不同)
    ↓
关键核心信息 (我们诉求什么)
    ↓
证明论据/支撑点 (客户为何相信)
```

## 核心陈述模板

### 使命陈述
```
我们通过 [方法] 为 [受众] 提供 [行动]，以便他们能够 [达成成果]。
```

### 愿景陈述
```
一个 [我们渴望看到的变化/愿景] 的世界。
```

### 价值主张
```
对于有 [需求/问题] 的 [目标客户]，
[产品/品牌] 是一款 [品类/类别]
能够 [带来核心利益/解决关键问题]。
不同于 [竞争对手]，
我们 [具备独特差异化优势]。
```

### 定位陈述
```
[品牌] 是专为 [受众] 打造的 [品类/类别]
满足他们期望获得 [期望达成的成果]
因为 [令其信服的支撑论据]。
```

## 核心信息架构

### 主要核心信息 (Primary Message)
用一句话概括你的核心价值。

### 辅助支撑信息 (Supporting Messages) (3-5)
每一条对应不同的利益点或受众需求。

| 核心信息 | 受众需求 | 证明论据/支撑点 |
|---------|---------------|-------------|
| [核心信息 1] | [需求] | [证明论据] |
| [核心信息 2] | [需求] | [证明论据] |
| [核心信息 3] | [需求] | [证明论据] |

### 电梯演讲 (Elevator Pitches)

**10 秒版本：**
[一句话激发兴趣]

**30 秒版本：**
[发现问题 + 解决方案 + 差异化特点]

**60 秒版本：**
[包含完整证明论据的全面推介]

## 核心信息检验

1. 表达是否清晰？（无晦涩术语）
2. 是否具备差异化？（竞品无法模仿）
3. 是否可信？（我们能否证明它）
4. 是否具有吸引力？（受众是否关心）
5. 是否保持一致？（是否符合品牌调性）

# 品牌一致性检查表

## 视觉一致性

### 标志 (Logo)
- [ ] 使用了正确的标志版本
- [ ] 保持了适当的安全空间/留白
- [ ] 仅使用经批准的颜色
- [ ] 在所有尺寸下均清晰可读
- [ ] 无任何未经授权的修改

### 颜色 (Colors)
- [ ] 仅使用品牌调色板颜色
- [ ] 颜色应用一致
- [ ] 满足无障碍设计的适当对比度
- [ ] 保持了颜色比例

### 排版与字体 (Typography)
- [ ] 使用品牌字体
- [ ] 正确的字重/样式
- [ ] 适当的层级结构
- [ ] 格式一致

### 图像与图片 (Imagery)
- [ ] 符合品牌风格
- [ ] 调色/滤镜一致
- [ ] 主题合适
- [ ] 达到质量标准

## 语气语调一致性

### 语气 (Tone)
- [ ] 符合品牌个性
- [ ] 符合语境
- [ ] 跨渠道一致
- [ ] 无冲突的信息

### 语言与用词 (Language)
- [ ] 使用品牌专属术语
- [ ] 大小写一致
- [ ] 缩写适当
- [ ] 行业术语级别适当

### 核心信息 (Messaging)
- [ ] 与关键信息保持一致
- [ ] 价值主张清晰
- [ ] 突显差异化特征
- [ ] 呼吁操作 (CTA) 一致

## 渠道审计

### 网站
- [ ] 首页
- [ ] 产品页面
- [ ] 博客/内容
- [ ] 页脚/导航栏

### 社交媒体
- [ ] 头像/资料图片
- [ ] 封面图片
- [ ] 个人简介/关于部分
- [ ] 帖子模板

### 电子邮件
- [ ] 页眉/页脚
- [ ] 模板
- [ ] 签名
- [ ] 自动发送的消息

### 宣传材料 (Collateral)
- [ ] 演示文稿/幻灯片
- [ ] 单页宣传册
- [ ] 名片
- [ ] 促销材料

## 常见问题

| 问题 | 解决方法 |
|-------|-----|
| 过期的标志 | 替换为当前版本 |
| 不符合品牌的颜色 | 更新为品牌调色板颜色 |
| 错误的字体 | 替换为品牌字体 |
| 语气语调不一致 | 应用风格指南 |
| 信息混乱/不一致 | 对齐到品牌核心信息框架 |

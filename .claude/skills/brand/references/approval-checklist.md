# 资产审批检查表

在审批之前审查市场营销资产的全面检查表。

## 快速审查

进行详细审查前，验证以下内容：
- [ ] 资产符合既定用途
- [ ] 适合目标受众
- [ ] 无明显错误或问题
- [ ] 符合营销活动目标

## 视觉元素

### 标志 (Logo) 使用规范
- [ ] 根据具体语境使用正确的标志变体
- [ ] 保持了适当的安全空间/留白
- [ ] 满足最小尺寸要求
- [ ] 仅使用批准的颜色
- [ ] 无任何未经授权的修改
- [ ] 在当前背景下显示效果合适

### 颜色合规性
- [ ] 仅使用品牌调色板颜色
- [ ] 主/辅/强调色比例适当 (60/30/10)
- [ ] 正确使用语义色
- [ ] 未引入任何非品牌颜色
- [ ] 所有元素之间保持一致

### 排版与字体
- [ ] 全文使用品牌字体
- [ ] 应用了正确的字重
- [ ] 具备良好的字号层级结构
- [ ] 尺寸适合该媒介渠道
- [ ] 行高充足
- [ ] 正文文本中没有孤行/寡行 (Orphans/Widows)

### 图像与图片
- [ ] 符合品牌摄影风格
- [ ] 主题/内容合适
- [ ] 质量达到要求
- [ ] 获得适当授权/注明出处
- [ ] 针对预期用途进行了优化

## 无障碍性 (Accessibility)

### 视觉无障碍
- [ ] 文本对比度 >= 4.5:1 (AA)
- [ ] 大号文本对比度 >= 3:1
- [ ] 交互元素有清晰可见的聚焦 (Focus) 状态
- [ ] 颜色并非传递信息的唯一指标
- [ ] 为所有图像添加 Alt 替代文本

### 内容无障碍
- [ ] 布局清晰、便于扫读
- [ ] 字号清晰易读
- [ ] 阅读顺序符合逻辑
- [ ] 标题结构有实际逻辑意义
- [ ] 链接文本描述了其目标指向

## 内容质量

### 文案审查
- [ ] 符合品牌语气
- [ ] 语境语调合适
- [ ] 未使用禁用词汇
- [ ] 价值主张清晰
- [ ] CTA 清晰且具吸引力
- [ ] 已进行校对，确认无误

### 信息传递
- [ ] 与核心关键信息对齐
- [ ] 突显差异化优势
- [ ] 强调“带来的利益”而非单纯的“功能特性”
- [ ] 明确面向目标受众
- [ ] 无冲突/矛盾的表述

## 技术要求

### 文件规范
- [ ] 文件格式正确
- [ ] 分辨率合适
- [ ] 文件大小已优化
- [ ] 符合命名规范
- [ ] 包含元数据

## 法律与合规

### 知识产权 (IP)
- [ ] 素材库图片已获得授权
- [ ] 音乐/音频版权已清理完毕
- [ ] 无商标侵权行为
- [ ] 用户生成内容 (UGC) 已获授权
- [ ] 在需要的地方注明了出处/致谢

### 行业监管
- [ ] 包含必要的免责声明/信息披露
- [ ] 无误导性陈述
- [ ] 定价准确无误
- [ ] 在需要的地方链接了相关条款
- [ ] 符合隐私政策规范

## 常见问题与解决方法

| 问题 | 解决方法 |
|-------|-----|
| 标志太小 | 放大至最小尺寸 |
| 错误的字体 | 替换为品牌字体 |
| 对比度过低 | 调整颜色以满足无障碍性要求 |
| 非品牌颜色 | 替换为调色板中的颜色 |
| 图像模糊 | 使用更高分辨率的源文件 |
| 缺失 Alt 文本 | 添加描述性 Alt 替代文本 |
| CTA 引导力度弱 | 强化行动导向的文案 |

## 自动化支持

`validate-asset.cjs` 脚本可以自动检查：
- 调色板合规性
- 最小尺寸
- 文件格式/大小
- 命名规范
- 基础元数据

运行：`node .claude/skills/brand/scripts/validate-asset.cjs <asset-path>`
